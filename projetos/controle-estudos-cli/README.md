# Controle de Estudos CLI

Micro-projeto em Python para registrar sessões de estudo pelo terminal.

## Versão atual

v0.1 - Registro, listagem e resumo de sessões usando listas, dicionários, funções e arquivo JSON.

## Funcionalidades

- Registrar sessão de estudo
- Validar dados de entrada
- Listar sessões salvas
- Mostrar resumo geral
- Persistir dados em arquivo JSON

## Conceitos praticados

- Funções
- Listas
- Dicionários
- Arquivos
- JSON
- Loops
- Condicionais
- Validação de dados
- Organização procedural
