# ads-prep-2026

Repositório de preparação técnica para Análise e Desenvolvimento de Sistemas, com foco em backend, fundamentos de computação e construção progressiva de projetos.

## Objetivo

Registrar estudos, exercícios, anotações e micro-projetos desenvolvidos durante o período pré-Fatec.

A proposta deste repositório é consolidar fundamentos técnicos, criar consistência de prática e formar uma base pública de evolução em programação.

## Trilhas de estudo

### Fundamentos

Estudos de apoio para operar melhor como desenvolvedor.

Conteúdos principais:

- Git e GitHub
- Terminal Linux
- Sistemas operacionais
- Redes e HTTP
- Lógica computacional
- Metodologias de produção

### Python

Trilha principal de programação durante a preparação.

Conteúdos principais:

- Sintaxe da linguagem
- Funções
- Estruturas de dados básicas
- Orientação a objetos
- Tratamento de erros
- Manipulação de arquivos
- Ambientes virtuais
- Testes iniciais

### SQL e banco de dados

Trilha de banco de dados relacional.

Conteúdos principais:

- SQL
- MySQL
- Modelagem relacional
- Consultas
- Filtros
- Joins
- Relacionamentos entre tabelas
- Integração entre aplicação e banco de dados

### Projetos

Área destinada a exercícios aplicados, micro-projetos e protótipos.

Projetos maiores poderão ter repositórios próprios quando tiverem escopo, estrutura e documentação suficientes.

## Foco técnico inicial

O foco inicial deste repositório é desenvolver base prática em:

- Python
- Git e GitHub
- Linux
- SQL com MySQL
- Fundamentos de backend
- Redes aplicadas a APIs
- Organização de projetos

## Metodologia

O estudo será conduzido por ciclos pequenos de prática.

Cada ciclo deve produzir algum registro concreto, como:

- exercício resolvido
- anotação técnica
- script funcional
- consulta SQL
- melhoria de código
- documentação
- micro-projeto

A prioridade é praticar, registrar e evoluir de forma contínua.
